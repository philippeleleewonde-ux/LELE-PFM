## Anti-Patterns Financiers

### Calculs
| Anti-pattern | Solution |
|--------------|----------|
| Valeurs hardcodées | Constantes nommées configurables |
| Erreurs floating point | Utiliser Decimal.js |
| Pas de check null/undefined | Validation systématique |
| Arrondis incohérents | Logique centralisée |

### Affichage
| Anti-pattern | Solution |
|--------------|----------|
| Chiffres bruts | `Intl.NumberFormat('fr-FR')` |
| Colonnes désalignées | `tabular-nums` + `text-right` |
| Pas de séparateurs | Formater: 1 234 567,89 € |
| Trop de décimales | 2 max pour montants |

### Extraction de formules
```python
from openpyxl import load_workbook

def extract_formulas(file_path: str) -> dict:
    """Extrait toutes les formules d'un fichier Excel"""
    wb = load_workbook(file_path)
    formulas = {}

    for sheet_name in wb.sheetnames:
        sheet = wb[sheet_name]
        sheet_formulas = {}

        for row in sheet.iter_rows():
            for cell in row:
                if cell.value and isinstance(cell.value, str):
                    if cell.value.startswith('='):
                        sheet_formulas[cell.coordinate] = {
                            'formula': cell.value,
                            'address': cell.coordinate
                        }

        if sheet_formulas:
            formulas[sheet_name] = sheet_formulas

    return formulas
```

### Comparaison de valeurs
```python
def compare_values(
    excel_value: float,
    platform_value: float,
    tolerance: float = 0.01
) -> dict:
    """Compare une valeur Excel avec la plateforme"""
    diff = abs(excel_value - platform_value)
    is_match = diff <= tolerance

    return {
        'excel': excel_value,
        'platform': platform_value,
        'difference': diff,
        'tolerance': tolerance,
        'match': is_match,
        'status': '✅ CONFORME' if is_match else '❌ ÉCART DÉTECTÉ'
    }
```
